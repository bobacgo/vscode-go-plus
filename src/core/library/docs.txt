在使用 go list -f 命令时，您可以在格式字符串中引用许多字段，以获取有关包或模块的特定信息。这些字段对应于 Go 内部的 Package 结构体的成员。以下是一些常用字段及其含义：
	•	{{.ImportPath}}：包的导入路径。
	•	{{.Name}}：包的名称。 ￼
	•	{{.Doc}}：包的文档字符串。
	•	{{.Dir}}：包所在的目录路径。
	•	{{.GoFiles}}：包中包含的 Go 源文件列表。
	•	{{.Deps}}：包的所有依赖项的导入路径列表。
	•	{{.Module}}：与包关联的模块信息，包含路径、版本等。

go list -f '{{.Module.Path}}' 获取当前包所属的模块名称
